# Smart-Healthcare-Monitoring-System

## Overview
The Smart Healthcare Monitoring System continuously monitors patient health parameters using IoT sensors and AI to provide early health alerts and remote monitoring.

## Features
- Heart rate monitoring
- Body temperature monitoring
- Oxygen level tracking
- AI health analysis
- Emergency alerts

## Technologies Used
- Python
- Streamlit
- IoT
- Machine Learning

## Future Scope
- Wearable device integration
- Hospital connectivity
- Cloud storage
- Predictive disease detection
